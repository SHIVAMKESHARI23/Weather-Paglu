A modern and responsive Weather Application built using React.js and Material UI. 

The app fetches real-time weather data using the OpenWeather API and dynamically updates the UI based on temperature and weather conditions. It features a glassmorphism design, animated gradient backgrounds, and smart theme switching (hot, cold, rainy).

Key Features:
- 🌤️ Real-time weather data fetching
- 🎨 Dynamic UI themes based on weather conditions
- 💎 Glassmorphism + modern UI design
- ⚡ Fast and responsive (React + Vite)
- 🔍 Search functionality for any city
- ❌ Error handling for invalid locations

Tech Stack:
- React.js
- Material UI (MUI)
- OpenWeather API
- CSS (Glassmorphism + Animations)
